=========================================
Survey User Input: Show Total Score Field
=========================================

* Shows the numeric total score field that is by default hidden (i.e. only
  the percentage score is shown)

Configuration
=============
* None needed

Usage
=====
* Open the participation form view, the Total Score is shown next to Score (%)
