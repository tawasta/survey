========================
survey matrix responsive
========================
* This module improves the responsiveness of survey matrix questions in Odoo. It ensures that the table adapts properly to mobile screens by restructuring the layout, hiding headers, and displaying column labels dynamically. The module inherits the matrix question template to include a data-label attribute, allowing better accessibility and usability on smaller screens.
